== Installation ==

= Installation =
1. In your WordPress Dashboard go to "Plugins" -> "Add Plugin".
2. Search for "Smart1Waze Floating Widget".
3. Install the plugin by pressing the "Install" button.
4. Activate the plugin by pressing the "Activate" button.
5. Go to "Smart1Waze" Settings section on the WordPress admin panel and enter valid api key.
6. Choose where you want to enable widget.
7. Options to enable/disable widget on custom post type, taxonomy
8. Save settings and widget will appear on your selected pages.


= Minimum Requirements =
* WordPress version 4.0 or greater.
* PHP version 5.2.4 or greater.
* MySQL version 5.0 or greater.

= Recommended Requirements =
* The latest WordPress version.
* PHP version 7.0 or greater.
* MySQL version 5.5 or greater.
